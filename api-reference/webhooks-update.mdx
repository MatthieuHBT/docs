---
title: Activer / désactiver un webhook
api: PATCH https://roapix.up.railway.app/api/v1/webhooks/{id}
description: Active ou désactive un webhook sans le supprimer.
---

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField path="id" type="string" required>UUID du webhook</ParamField>

<ParamField body="is_active" type="boolean" required>
  `true` pour activer, `false` pour désactiver
</ParamField>

<RequestExample>
```bash Désactiver
curl -X PATCH "https://roapix.up.railway.app/api/v1/webhooks/wh-uuid-..." \
  -H "Authorization: Bearer roapix_votre_clé" \
  -H "Content-Type: application/json" \
  -d '{"is_active": false}'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": {
    "id": "wh-uuid-...",
    "url": "https://hook.make.com/abc123",
    "events": ["sync.complete"],
    "is_active": false,
    "created_at": "2026-03-23T14:00:00.000Z"
  },
  "workspace_id": "a1b2c3d4-...",
  "generated_at": "2026-03-23T20:00:00.000Z"
}
```
</ResponseExample>
