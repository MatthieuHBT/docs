---
title: Créer une alerte
api: POST https://roapix.up.railway.app/api/v1/alerts
description: Crée une nouvelle règle d'alerte pour votre workspace.
---

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField body="alert_type" type="string" required>
  Type d'alerte : `roas_below`, `spend_above`, `daily_report`, `budget_exceeded`
</ParamField>

<ParamField body="threshold" type="number">
  Seuil déclencheur (ex : `2.0` pour ROAS < 2x)
</ParamField>

<ParamField body="channels" type="array">
  Canaux de notification : `["slack"]`, `["email"]` ou `["slack", "email"]`. Défaut : `["slack"]`
</ParamField>

<ParamField body="days_of_week" type="array">
  Jours d'envoi (0=dim, 1=lun … 6=sam). `null` = tous les jours.
</ParamField>

<ParamField body="send_hour" type="number">
  Heure d'envoi (0–23). Défaut : `9`
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://roapix.up.railway.app/api/v1/alerts \
  -H "Authorization: Bearer roapix_votre_clé" \
  -H "Content-Type: application/json" \
  -d '{"alert_type": "roas_below", "threshold": 2.0, "channels": ["slack", "email"]}'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "data": {
    "id": "alert-uuid-...",
    "alert_type": "roas_below",
    "threshold": 2.0,
    "channels": ["slack", "email"],
    "days_of_week": null,
    "send_hour": 9,
    "is_active": true,
    "created_at": "2026-03-23T20:00:00.000Z"
  },
  "workspace_id": "a1b2c3d4-...",
  "generated_at": "2026-03-23T20:00:00.000Z"
}
```
</ResponseExample>
