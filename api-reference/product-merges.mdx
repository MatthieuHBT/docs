---
title: Fusions de produits
api: GET https://roapix.up.railway.app/api/v1/product-merges
description: Liste les alias de produits configurés — utile pour comprendre comment les noms sont normalisés.
---

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

Quand plusieurs noms de campagnes font référence au même produit (ex : `Felipet Cat FR`, `felipet-cat`, `FC`), les fusions de produits les unifient sous un nom canonique.

## Réponse

<ResponseField name="data" type="array">
  <Expandable title="Propriétés de chaque fusion">
    <ResponseField name="alias" type="string">Nom de campagne d'origine</ResponseField>
    <ResponseField name="canonical" type="string">Nom canonique unifié</ResponseField>
    <ResponseField name="created_at" type="string">Date de création</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl https://roapix.up.railway.app/api/v1/product-merges \
  -H "Authorization: Bearer roapix_votre_clé"
```
</RequestExample>
