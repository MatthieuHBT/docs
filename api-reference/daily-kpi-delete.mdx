---
title: Supprimer une entrée KPI
api: DELETE https://roapix.up.railway.app/api/v1/daily-kpi/:id
description: Supprime définitivement une entrée KPI journalière.
---

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField path="id" type="string" required>
  UUID de l'entrée à supprimer
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://roapix.up.railway.app/api/v1/daily-kpi/uuid-..." \
  -H "Authorization: Bearer roapix_votre_clé"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": { "deleted": true, "id": "uuid-..." },
  "workspace_id": "a1b2c3d4-...",
  "generated_at": "2026-03-23T20:00:00.000Z"
}
```
</ResponseExample>
