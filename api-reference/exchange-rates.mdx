---
title: Taux de change
api: GET https://roapix.up.railway.app/api/v1/exchange-rates
description: Retourne les taux de change configurés pour la conversion vers USD.
---

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

## Réponse

<ResponseField name="data" type="array">
  <Expandable title="Propriétés de chaque taux">
    <ResponseField name="currency" type="string">Code devise (ex : `EUR`, `GBP`)</ResponseField>
    <ResponseField name="rate_to_usd" type="number">Taux de conversion vers USD</ResponseField>
    <ResponseField name="fetched_at" type="string | null">Date de dernière mise à jour</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl https://roapix.up.railway.app/api/v1/exchange-rates \
  -H "Authorization: Bearer roapix_votre_clé"
```
</RequestExample>
