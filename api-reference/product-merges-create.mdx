---
title: Créer une fusion
api: POST https://roapix.up.railway.app/api/v1/product-merges
description: Associe un alias de produit à un nom canonique.
---

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField body="alias" type="string" required>
  Nom de campagne à normaliser (ex : `FC`, `felipet-cat`)
</ParamField>

<ParamField body="canonical" type="string" required>
  Nom canonique cible (ex : `Felipet Cat`)
</ParamField>

<Tip>
  Si l'alias existe déjà, son canonical sera mis à jour.
</Tip>

<RequestExample>
```bash cURL
curl -X POST https://roapix.up.railway.app/api/v1/product-merges \
  -H "Authorization: Bearer roapix_votre_clé" \
  -H "Content-Type: application/json" \
  -d '{"alias": "FC", "canonical": "Felipet Cat"}'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "data": {
    "alias": "FC",
    "canonical": "Felipet Cat",
    "created_at": "2026-03-23T20:00:00.000Z"
  },
  "workspace_id": "a1b2c3d4-...",
  "generated_at": "2026-03-23T20:00:00.000Z"
}
```
</ResponseExample>
