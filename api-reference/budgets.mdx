## Lister les budgets

<api>GET https://roapix.up.railway.app/api/v1/budgets</api>

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

Cet endpoint ne prend pas de paramètres de date — il retourne les budgets actuellement configurés.

## Réponse

<ResponseField name="data" type="array">
  Liste des campagnes avec leur budget journalier.
  <Expandable title="Propriétés de chaque item">
    <ResponseField name="campaign_id" type="string">Identifiant Meta de la campagne</ResponseField>
    <ResponseField name="campaign_name" type="string">Nom de la campagne</ResponseField>
    <ResponseField name="daily_budget" type="number">Budget journalier en USD</ResponseField>
    <ResponseField name="account" type="string">Nom du compte publicitaire</ResponseField>
    <ResponseField name="product" type="string">Produit associé</ResponseField>
    <ResponseField name="market" type="string">Marché (code pays)</ResponseField>
    <ResponseField name="status" type="string">Statut de la campagne (`ACTIVE`, `PAUSED`, etc.)</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl https://roapix.up.railway.app/api/v1/budgets \
  -H "Authorization: Bearer roapix_votre_clé"
```

```python Python
import requests

r = requests.get(
    "https://roapix.up.railway.app/api/v1/budgets",
    headers={"Authorization": "Bearer roapix_votre_clé"}
)
budgets = r.json()["data"]
total = sum(b["daily_budget"] for b in budgets if b["status"] == "ACTIVE")
print(f"Total budget journalier actif : ${total:,.2f}")
```

```javascript JavaScript
const res = await fetch('https://roapix.up.railway.app/api/v1/budgets', {
  headers: { 'Authorization': 'Bearer roapix_votre_clé' }
})
const { data } = await res.json()
const active = data.filter(b => b.status === 'ACTIVE')
console.log(`${active.length} campagnes actives`)
```

## Modifier le budget d'une campagne

<api>PATCH https://roapix.up.railway.app/api/v1/budgets/:campaignId</api>

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField path="campaignId" type="string" required>
  ID Meta de la campagne (ex : `120213456789`)
</ParamField>

<ParamField body="daily_budget" type="number" required>
  Nouveau budget journalier en USD
</ParamField>

<RequestExample>
```bash cURL
curl -X PATCH "https://roapix.up.railway.app/api/v1/budgets/120213456789" \
  -H "Authorization: Bearer roapix_votre_clé" \
  -H "Content-Type: application/json" \
  -d '{"daily_budget": 350.00}'
```

```python Python
import requests

r = requests.patch(
    "https://roapix.up.railway.app/api/v1/budgets/120213456789",
    json={"daily_budget": 350.00},
    headers={"Authorization": "Bearer roapix_votre_clé"}
)
print(r.json()["data"])
```
</RequestExample>

<ResponseExample>

## Supprimer une campagne des budgets

<api>DELETE https://roapix.up.railway.app/api/v1/budgets/:campaignId</api>

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField path="campaignId" type="string" required>
  ID Meta de la campagne à supprimer
</ParamField>

<Warning>
  La suppression retire la campagne de la base Roapix. Elle sera recréée au prochain sync Meta.
</Warning>

<RequestExample>
```bash cURL
curl -X DELETE "https://roapix.up.railway.app/api/v1/budgets/120213456789" \
  -H "Authorization: Bearer roapix_votre_clé"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "data": { "deleted": true, "campaign_id": "120213456789" },
  "workspace_id": "a1b2c3d4-...",
  "generated_at": "2026-03-23T20:00:00.000Z"
}
```
</ResponseExample>
