---
title: Modifier une alerte
api: PATCH https://roapix.up.railway.app/api/v1/alerts/:id
description: Modifie le seuil, les canaux ou l'état d'une règle d'alerte.
---

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField path="id" type="string" required>UUID de la règle d'alerte</ParamField>

<ParamField body="threshold" type="number">Nouveau seuil déclencheur</ParamField>
<ParamField body="channels" type="array">Canaux : `["slack"]`, `["email"]`, `["slack", "email"]`</ParamField>
<ParamField body="days_of_week" type="array">Jours d'envoi (0–6)</ParamField>
<ParamField body="send_hour" type="number">Heure d'envoi (0–23)</ParamField>
<ParamField body="is_active" type="boolean">Activer ou désactiver</ParamField>

<RequestExample>
```bash Désactiver une alerte
curl -X PATCH "https://roapix.up.railway.app/api/v1/alerts/alert-uuid-..." \
  -H "Authorization: Bearer roapix_votre_clé" \
  -H "Content-Type: application/json" \
  -d '{"is_active": false}'
```

```bash Changer le seuil ROAS
curl -X PATCH "https://roapix.up.railway.app/api/v1/alerts/alert-uuid-..." \
  -H "Authorization: Bearer roapix_votre_clé" \
  -H "Content-Type: application/json" \
  -d '{"threshold": 2.5}'
```
</RequestExample>
