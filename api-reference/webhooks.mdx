Les webhooks permettent à Roapix d'appeler votre URL externe automatiquement après un événement — sync Meta terminé, alerte déclenchée, etc.

## Vérifier la signature

Chaque requête inclut `X-Roapix-Signature` — HMAC-SHA256 du body signé avec l'ID du webhook :

```python
import hmac, hashlib

def verify(body: bytes, signature: str, webhook_id: str) -> bool:
    expected = hmac.new(webhook_id.encode(), body, hashlib.sha256).hexdigest()
    return hmac.compare_digest(expected, signature)
```
