---
title: Webhooks
description: Recevez des notifications en temps réel après chaque sync ou alerte.
---

Les webhooks permettent à Roapix d'appeler votre URL externe automatiquement après un événement — sync Meta terminé, alerte déclenchée, etc. Idéal pour piloter des workflows Make/Zapier ou des bots Slack sans polling.

## Événements disponibles

| Événement | Déclenché quand |
|-----------|----------------|
| `sync.complete` | Un sync Meta se termine avec succès |
| `alert.fired` | Une alerte (ROAS, budget) se déclenche |
| `budget.exceeded` | Un budget journalier est dépassé |

## Format du payload

Chaque requête POST vers votre URL contient :

```json
{
  "event": "sync.complete",
  "workspace_id": "a1b2c3d4-...",
  "fired_at": "2026-03-23T14:00:00.000Z",
  "data": {
    "campaigns_count": 142,
    "range": { "since": "2026-03-23", "until": "2026-03-23" }
  }
}
```

### Vérifier la signature

Chaque requête inclut un header `X-Roapix-Signature` — HMAC-SHA256 du body signé avec l'ID du webhook :

```python
import hmac, hashlib

def verify(body: bytes, signature: str, webhook_id: str) -> bool:
    expected = hmac.new(webhook_id.encode(), body, hashlib.sha256).hexdigest()
    return hmac.compare_digest(expected, signature)
```

## Enregistrer un webhook

<api>POST https://roapix.up.railway.app/api/v1/webhooks</api>

<ParamField header="Authorization" type="string" required>
  `Bearer roapix_votre_clé`
</ParamField>

<ParamField body="url" type="string" required>
  URL HTTPS vers laquelle Roapix enverra les événements
</ParamField>

<ParamField body="events" type="array">
  Liste des événements à écouter. Par défaut : tous les événements.
  Valeurs : `sync.complete`, `alert.fired`, `budget.exceeded`
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://roapix.up.railway.app/api/v1/webhooks \
  -H "Authorization: Bearer roapix_votre_clé" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://hook.make.com/abc123", "events": ["sync.complete"]}'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "data": {
    "id": "wh-uuid-...",
    "url": "https://hook.make.com/abc123",
    "events": ["sync.complete"],
    "is_active": true,
    "created_at": "2026-03-23T14:00:00.000Z"
  },
  "workspace_id": "a1b2c3d4-...",
  "generated_at": "2026-03-23T14:00:00.000Z"
}
```
</ResponseExample>
