## 1. Générer une clé API

1. Connectez-vous sur [roapix.up.railway.app](https://roapix.up.railway.app)
2. Allez dans **Paramètres → API Keys**
3. Cliquez **Générer**, nommez la clé (ex : "Test")
4. **Copiez immédiatement** la clé — elle ne sera plus affichée

## 2. Tester la connexion

```bash
curl https://roapix.up.railway.app/api/v1/me \
  -H "Authorization: Bearer roapix_votre_clé"
```

Réponse attendue :

```json
{
  "data": {
    "workspace_id": "a1b2c3d4-...",
    "workspace_name": "Votre Workspace",
    "api_version": "v1"
  }
}
```

## 3. Récupérer vos performances d'hier

```bash
curl "https://roapix.up.railway.app/api/v1/summary?date_preset=yesterday" \
  -H "Authorization: Bearer roapix_votre_clé"
```

```python Python
import requests

r = requests.get(
    "https://roapix.up.railway.app/api/v1/summary",
    params={"date_preset": "yesterday"},
    headers={"Authorization": "Bearer roapix_votre_clé"}
)
d = r.json()["data"]
print(f"Spend: ${d['spend']:,.0f} | Revenue: ${d['revenue']:,.0f} | ROAS: {d['roas']}x")
```

## 4. Paramètres de date

Tous les endpoints acceptent ces raccourcis :

| `date_preset` | Période |
|--------------|---------|
| `today` | Aujourd'hui |
| `yesterday` | Hier |
| `last_7d` | 7 derniers jours |
| `last_30d` | 30 derniers jours |

Ou des dates précises avec `?since=2026-03-01&until=2026-03-23`.
