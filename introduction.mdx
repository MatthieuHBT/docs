## Base URL

```
https://roapix.up.railway.app/api/v1
```

Chaque requête doit inclure le header `Authorization: Bearer roapix_xxx`.

## Réponse standard

```json
{
  "data": { ... },
  "workspace_id": "a1b2c3d4-...",
  "generated_at": "2026-03-23T14:00:00.000Z"
}
```

## Exemple

```bash
curl "https://roapix.up.railway.app/api/v1/summary?date_preset=last_7d" \
  -H "Authorization: Bearer roapix_votre_clé"
```

```json
{
  "data": {
    "since": "2026-03-17",
    "until": "2026-03-23",
    "spend": 12450.00,
    "revenue": 38200.00,
    "roas": 3.07,
    "orders": 312,
    "top_ad": { "name": "Felipet Cat — FR — V3", "spend": 1820.00 }
  }
}
```

## Limites

| Limite | Valeur |
|--------|--------|
| Requêtes / minute | 60 par clé API |
| Résultats campaigns | max 2 000 |
| Résultats winners | max 1 000 |
| Historique sync | max 100 entrées |
