## Base URL

```
https://roapix.up.railway.app/api/v1
```

## Format des réponses

Toutes les réponses suivent la même enveloppe JSON :

```json
{
  "data": { ... },
  "workspace_id": "uuid-de-votre-workspace",
  "generated_at": "2026-03-23T14:00:00.000Z"
}
```

## Ce que vous pouvez faire

| Catégorie | Endpoints |
|-----------|-----------|
| Workspace | Infos, résumé global |
| Publicité | Dépenses, budgets, meilleures pubs, comptes, produits |
| Revenus | Commandes et CA Shopify |
| KPI équipe | Lecture et écriture des entrées journalières |
| Webhooks | Notifications push après chaque sync |
