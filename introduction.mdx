## Base URL

```
https://roapix.up.railway.app/api/v1
```

## Format des réponses

Toutes les réponses sont en JSON et suivent cette structure :

```json
{
  "data": { ... },
  "workspace_id": "uuid-de-votre-workspace",
  "generated_at": "2026-03-23T14:00:00.000Z"
}
```

## Rate limiting

**60 requêtes par minute** par clé API. Les headers de réponse indiquent votre consommation :

| Header | Description |
|--------|-------------|
| `RateLimit-Limit` | Limite par fenêtre |
| `RateLimit-Remaining` | Requêtes restantes |
| `RateLimit-Reset` | Timestamp de reset |
